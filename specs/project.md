# Project: Matt Small's Landing Page

## Overview
Professional landing page for Matt Small, Principal Consultant specializing in FinOps, Enterprise AI adoption, and Cloud strategy.

## Goals
1. Generate consultation inquiries via email
2. Establish credibility through credentials
3. Showcase expertise in FinOps, AI, and Cloud
4. Build trust with professional presentation

## Target Audience
- Enterprise decision-makers (Fortune 500 executives, CTOs, CIOs)
- Startup founders
- Technology leaders

## Value Proposition
"Catalyst for systemic and cultural change, driving transformation that improves companies from the inside out."

## Scope
**In Scope**: Single landing page, professional introduction, credibility badges, email contact, responsive design, WCAG AA compliance

**Out of Scope**: Multi-page site, blog, contact forms with backend, user auth, e-commerce, analytics dashboard

## Technical Stack
- HTML5, CSS3 (inline)
- GitHub Pages deployment
- No build process
